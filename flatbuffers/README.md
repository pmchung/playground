## Flatbuffers
___

## Build `flatc` executable


```
brew install cmake`
```
### Clone flatbuffers repository
```
https://github.com/google/flatbuffers
```
### Build
```
cmake -G "Unix Makefiles"
cmake -G "Visual Studio 10"
cmake -G "Xcode"
```
```
make
```
./profile
```
alias flatc="PATH/TO/flatbuffers/flatc"
```
